Data Visualization

Dividing the Sky

This project came about while reading Alexander Boxer's "A Scheme of Heaven", a fascinating book about the history of astrology. As I was reading, I wanted to keep all the concepts I was learning about straight in my head and practice using the p5.js graphics library for data visualization. All I got around to was showing which signs were associated with which ranges of ecliptic latitude, but perhaps I'll return to it some day.

Controls:
Up/Down - Zoom in/out
Left/Right - Orbit around
z/x - Orbit up/down
K - Enable/disable auto-orbit
