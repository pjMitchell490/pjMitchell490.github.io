Data Visualization

Dividing the Sky

This project came about while reading Alexander Boxer's "A Scheme of Heaven", a fascinating book about the history of astrology. As I was reading, I wanted to keep all the concepts I was learning about straight in my head and practice using the p5.js graphics library for data visualization. There are more features I'd like to add some day, including the planets, a view looking up from a given latitude on Earth, and estimating the positions of bodies at a provided date and time.

Controls:
P - Pause
Up/Down - Zoom in/out
Left/Right - Orbit around
z/x - Orbit up/down
K - Enable/disable auto-orbit
